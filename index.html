Bienvenue ! Nous sommes un groupe de recherche explorant la propagation des ondes élastiques dans une large gamme de matériaux non conventionnels, notamment les structures souples, les milieux granulaires, les réseaux topologiques et les matériaux non linéaires. Notre travail se concentre sur les systèmes modèles pour observer des phénomènes ondulatoires non linéaires frappants, tels que les bifurcations, la propagation de solitons, les vagues de type "rogue" et les vagues de transition de type "domino". Inversement, nous concevons des structures fonctionnelles visant à contrôler la propagation des ondes élastiques pour des applications telles que le filtrage, l'atténuation, le guidage ou la récolte d'énergie.
